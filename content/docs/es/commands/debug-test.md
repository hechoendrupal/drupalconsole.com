# debug:test
Ofrece un listado de Tests Unitarios disponibles en la aplicación.

**Uso:**
```
drupal debug:test [arguments] [options]
td
```

## Opciones disponibles
Opción | Detalles
-------|-------------
--test-class | Clase de Test

## Argumentos disponibles
Argumento | Detalles
---------|-------------
group | Grupo

## Ejemplos
* 
```
drupal debug:test
```
