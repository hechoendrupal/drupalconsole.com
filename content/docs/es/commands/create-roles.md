# create:roles
Crea roles de ejemplo para su aplicación Drupal 8.

**Uso:**
```
drupal create:roles [options]
crr
```

## Opciones disponibles
Opción | Detalles
-------|-------------
--limit | Cuántos roles le gustaría crear

## Ejemplos
* Facilitar roles
```
drupal create:roles
```
* Facilitar el número de roles a crear
```
drupal create:roles
```
