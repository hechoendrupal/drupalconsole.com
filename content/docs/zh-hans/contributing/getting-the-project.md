# 获取项目

## Fork
Fork [Drupal Console](https://github.com/hechoendrupal/drupal-console/fork) 到你自己的账号

## 克隆
克隆 Fork 后的版本到本地
```
$ git clone git@github.com:[你的github用户账号]/drupal-console.git
```

## 安装依赖
克隆完项目，你需要使用 Composer 下载依赖

```
$ cd /path/to/drupal-console
$ composer install
```
